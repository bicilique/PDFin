# PDFin - Feature Illustrations

## Images Location
This directory contains feature illustrations for the tool tiles displayed on the Home page.

## Required Images (80x80px PNG format):
1. **merge.png** - Illustration for Merge PDF feature
2. **split.png** - Illustration for Split PDF feature  
3. **compress.png** - Illustration for Compress PDF feature
4. **protect.png** - Illustration for Lock/Protect PDF feature

## Recommended Sources (Free, No Attribution Required):

### unDraw (https://undraw.co/)
- License: Free to use without attribution
- Customizable SVG illustrations that can be exported as PNG
- Recommended searches:
  - Merge: "files", "documents", "together"
  - Split: "split", "divide", "separate"
  - Compress: "compress", "reduce", "optimize"
  - Protect: "security", "lock", "password"

### ManyPixels Gallery (https://www.manypixels.co/gallery)
- License: Free to use without attribution
- High-quality illustrations in PNG format
- Search for document-related illustrations

## How to Add Images:

1. Visit unDraw.co or ManyPixels gallery
2. Search for relevant illustration matching each feature
3. Download as PNG (or SVG â†’ export to PNG)
4. Resize to 80x80px or similar (JavaFX will preserve aspect ratio)
5. Save with the exact filenames listed above in this directory

## Color Customization:
When downloading from unDraw, you can customize the primary color to match PDFin's brand:
- Primary Color: #5b65ea (indigo/blue)
- Accent Color: #1a7f64 (teal/green)

## Alternative: Use Placeholder Images
For testing, you can use simple colored shapes or emoji-based placeholders until final illustrations are ready.
